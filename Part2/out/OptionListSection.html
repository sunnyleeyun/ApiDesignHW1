Class: OptionListSection

OptionListSection()

An OptionList section includes a table that presents the provided option definitions.

Constructor

new OptionListSection()

Properties:
Name Type Attributes Description
header string The section header. The style of the output is always bold and underlined.
optionList Array.<OptionDefinition> The list of options to be displayed. An array of option definition objects.
  • description (string): A description of the option.
  • typeLabel (string): A custom type label to replace the default type string (e.g., `underline file`). More descriptive type labels like `ms`, `file`, `command`, etc., are often more useful.
superGroup string | Array.<string> <optional>
 Options to be grouped. If specified, only options from this particular group will be printed. The "super" prefix is used to differentiate from "group" at a lower hierarchy.
hide string | Array.<string> <optional>
 Options to be hidden. Provide the names of one or more option definitions to hide them from the option list.
reverseNameOrder boolean <optional>
 Reverse the order of option name and alias. By default, this property is set to false. When set to true, the option alias will be displayed after the name, e.g., `--verbose, -v` instead of `-v, --verbose`).
tableOptions object <optional>
 The style of the option list output. This is an options object suitable for passing into table-layout, providing customization options for the option list table.

Examples
Example 1: Demonstrating a simple OptionList section.
const section1 = {
  header: 'Options',
  optionList: [
    {
      name: 'help',
      alias: 'h',
      description: 'Display this usage guide.'
    }
  ]
} 
Example 2: Illustrating the usage of a `superGroup` in an OptionList section.
const section2 = {
  header: 'Main options',
  optionList: [
    {
      name: 'help',
      description: 'Display this usage guide.',
      alias: 'h',
      type: Boolean,
      group: 'main'
    },
    {
      name: 'src',
      description: 'The input files to process',
      multiple: true,
      defaultOption: true,
      typeLabel: '{underline file} ...',
      group: 'input'
    },
    {
      name: 'timeout',
      description: 'Timeout value in ms',
      alias: 't',
      typeLabel: '{underline ms}',
      group: 'main'
    }
  ],
  superGroup: ['main', 'input']
}
Example 3: Illustrating the usage of a `hide` in an OptionList section.
const section3 = {

      type: String,
      multiple: true,
      defaultOption: true,
      typeLabel: '{underline file} ...'
    }
  ]
}
